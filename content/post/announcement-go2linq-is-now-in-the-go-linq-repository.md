I am happy to announce that my [go2linq](https://github.com/kalaninja/go2linq) (a powerful language integrated query (LINQ) library for Go) project has been merged into the [go-linq](https://github.com/ahmetalpbalkan/go-linq)'s master branch. Now it is available as the newest version of [go-linq](http://ahmetalpbalkan.github.io/go-linq/) library (v2.0.0). 
<!--more-->

All further development will be conducted in the new repository and go2linq will remain only for compatibility reasons. **Please, do not use it anymore!**

I would like to thank [Ahmet Alp Balkan](https://ahmetalpbalkan.com/) who has made me the maintainer of the go-linq repository and is cooperating in the further development.
